# Journal des changements

Toutes les modifications notables sont documentées ici. Le projet suit
[Semantic Versioning](https://semver.org/lang/fr/).

## [0.2.0] - 2026-07-18

### Ajouté

- suivi comptable des transactions avec filtres, pagination et rapprochement ;
- navigation entre le tableau de bord et le livre comptable ;
- pipeline CI de tests, lint, typage et build ;
- publication automatisée d'une GitHub Release à partir d'un tag versionné.

## [0.1.0] - 2026-07-18

### Ajouté

- socle monorepo API et interface web ;
- base PostgreSQL, authentification de démonstration et tableau de bord connecté.
